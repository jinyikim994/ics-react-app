## 📌 소개
이 프로젝트는 Iconic Creative Seoul의 브랜드 웹사이트를 React로 구현한 버전입니다.  
정적 사이트로 배포되며, 퍼블리싱 + React 컴포넌트 기반으로 구성되었습니다.

## 🧰 사용 기술
- React
- React Router 
- SCSS
